# Limitcycle

Limitcycle is the independent software effort of Ferhat Fidan — founder, sole developer, and inventor.

The name comes from dynamical systems: a limit cycle is a closed orbit that a system settles into and sustains on its own.

## JARVIS RD Assistant

[JARVIS RD Assistant](https://github.com/limitcycle-oss/jarvis-rd-assistant) is the first major public project of this org and is actively developed: a self-hosted workspace for scientific literature. Automated paper discovery, AI generated summaries, synthesis where every claim links back to its source, all known as retrieval augmented generation (RAG), PDF annotation, Zotero sync, project management, and spaced-repetition review for what you've read.

It's local-first. By default it runs Ollama models on your own hardware; cloud models are optional, routed through LiteLLM, and configurable by entering a working API key. Install is Docker-based, sign-in uses passkeys, and backup/restore is built in. Apache-2.0. [Documentation](https://limitcycle-oss.github.io/jarvis-rd-assistant/).
